# ReactJS GraphQL tutorial

A simple application build with React and Apollo client

## Stack
- React
- Apollo client
- Tailwind
- ExpressJS
- Apollo server

## How to use
**Backend**
```
cd backend
npm install
npm start
```
**Frontend**
```
cd frontend
npm install
npm start
open localhost:3000
```
